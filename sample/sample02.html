Sample 02

If you want to have two or more barenotes in one page, apply barenote to each tag like below Of course, you can use class instead of id.
See sample03..
    <script type='text/javascript'>
      $(document).ready(function() {
        var ModuleA = Barenote.apply($('#moduleA'));
        var ModuleB = Barenote.apply($('#moduleB'));
      });
    </script>
